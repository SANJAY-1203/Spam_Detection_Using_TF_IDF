# Email Spam Detection using NLP

## Overview
This project detects whether an email is Spam or Not Spam using Machine Learning and NLP techniques.

## Features
- Text preprocessing using NLTK
- TF-IDF vectorization
- Multinomial Naive Bayes classifier
- Flask web application for real-time prediction

## Dataset
- SMS Spam Collection Dataset (~5,500 messages)

- Accuracy: XX%

## Tech Stack
- Python, Scikit-learn, NLTK, Flask

2. Run app: python app.py
